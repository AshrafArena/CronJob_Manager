# Cron Job Manager

A Flask web application to manage user-based cron jobs.

## Features

- List existing cron jobs
- Add new cron jobs
- Delete cron jobs

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/ashrafarena/cronjob_manager.git
   cd cronjob_manager
